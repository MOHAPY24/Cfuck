# Cfuck
## A simple and minimalistic Brainfuck interpreter written in C.

### Features
- Lightweight and fast execution.
- Easy to compile and run on any system with a C compiler.
- Supports standard Brainfuck commands.
### Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/MOHAPY24/cfuck.git
   cd cfuck
   ```
2. Compile the interpreter:
   ```bash
   gcc -o cfuck cfuck.c
   ```
3. Run a Brainfuck program:
   ```bash
   ./cfuck
   ```
